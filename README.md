# LinguaBot 🌍

A language learning app that explores any word across 6 languages using Claude AI.

## Setup

1. Install dependencies:
```
npm install
```

2. Create a `.env` file:
```
ANTHROPIC_API_KEY=your_key_here
```

3. Run locally:
```
npm run dev
```

## Deploy to Vercel

1. Push this folder to GitHub
2. Import repo on vercel.com
3. Add environment variable: `ANTHROPIC_API_KEY`
4. Deploy!
